# Hi, I'm Abelardo Rodríguez-Pretelín 👋

**Researcher in computational hydrogeology** · Instituto de Geología, UNAM
Dr.-Ing., University of Stuttgart (2020)

I build **AI-driven, reproducible methods for groundwater protection and management under uncertainty** — deep-learning surrogates, Bayesian uncertainty quantification, generative geostatistics, and reinforcement learning for aquifer decision-making.

🔬 **Research line:** *Closed-loop groundwater management* — making the delineation of wellhead protection areas fast (surrogates), realistic (generative geology) and actionable (sequential design & control).

## Open scientific software

| Repository | What it is |
|---|---|
| [**rrwpt**](https://github.com/abelardo1685/rrwpt) | Transient probabilistic wellhead protection area delineation (transient FEM flow + reverse random-walk particle tracking + geostatistics). Validated Python package. |
| [**kfield-generator**](https://github.com/abelardo1685/kfield-generator) | Spectral generator of random hydraulic-conductivity fields for 3D aquifers. |
| [**pyOMOPSO**](https://github.com/abelardo1685/pyOMOPSO) | Parallel multi-objective particle swarm optimizer (OMOPSO). |
| [**unam-groundwater-optimization**](https://github.com/abelardo1685/unam-groundwater-optimization) | Graduate course: Optimization in Groundwater — UNAM. |
